# watadweb — a3mal watad

متجر إلكتروني لأزياء الرجال (Next.js) — عربي/إنجليزي، عملة عراقية، لوحة تحكم محلية.

## التشغيل محلياً

```bash
npm install
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000)

لوحة التحكم: `/admin` — كلمة المرور الافتراضية: `watad2026`

## البناء

```bash
npm run build
npm start
```
